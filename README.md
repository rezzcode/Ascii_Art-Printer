## Usage 

clone this repo:

move to the backend directory 

```
git clone https://learn.zone01kisumu.ke/git/elvisotieno0/ascii-art-web.git
cd ascii-art-web/backend
```

then run the go server

```
go run .
```

the website can be accessed by searching:

```
http://localhost:8000/
```

on your browser

## Technology

this project is made posible with:

- go            : packages : net/http, log, and encoding/json ; made the web functionality to be a success
- html          : used to stracture what would be rendered
- css           : used to style what is rendered
- js            : useful on creating a communication path between go server and what is being presented with html and css
